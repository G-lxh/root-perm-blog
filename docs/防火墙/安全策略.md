# 防火墙安全策略配置

## 安全策略概述

安全策略是防火墙的核心配置，用于控制网络流量的进出。

## 策略配置原则

### 1. 默认拒绝原则
默认情况下拒绝所有流量，只允许明确允许的流量。

### 2. 最小权限原则
只授予完成任务所需的最小权限。

### 3. 明确性原则
策略规则应该清晰明确。

## 策略配置示例

```
# 允许内网访问外网HTTP/HTTPS
rule permit source 192.168.1.0/24 destination any service http https

# 允许外网访问内网Web服务器
rule permit source any destination 192.168.1.100 service http https

# 拒绝所有其他流量
rule deny source any destination any
```

## NAT配置

```
# 源NAT - 内网访问外网
nat outbound source 192.168.1.0/24 address-group 202.100.1.1-202.100.1.10

# 目的NAT - 外网访问内网服务器
nat server protocol tcp global 202.100.1.5 80 inside 192.168.1.100 80
```
